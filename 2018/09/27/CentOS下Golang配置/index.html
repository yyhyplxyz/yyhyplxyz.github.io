安装完毕后，我们需要配置环境变量，令我们在终端中可以使用go命令。

首先我们要了解下go语言的工作空间。工作空间，应该由 /bin, /src/, /pkg 三个文件夹组成。三个文件夹的作用如下：

Folder Usage
bin 存放编译后的程序包
pkg 存放编译生成的对象文件
src 外部库/源文件
